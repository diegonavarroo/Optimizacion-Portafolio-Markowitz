# Optimizaci-n-Portafolio-Markowitz
Este proyecto implementa un modelo cuantitativo de optimización de portafolios de inversión basado en la Teoría Moderna de Portafolios . El objetivo principal es encontrar la asignación óptima de capital entre un conjunto de activos para maximizar el rendimiento esperado ajustado por riesgo, utilizando el  Ratio de Sharpe como métrica objetivo.
